Requirements
------------

Hard requirements
^^^^^^^^^^^^^^^^^

DeepChem currently supports Python 3.5 through 3.7 and requires these packages on any condition.

- `joblib`_
- `NumPy`_
- `pandas`_
- `scikit-learn`_
- `SciPy`_
- `TensorFlow`_

  - `deepchem>=2.4.0` requires tensorflow v2
  - `deepchem<2.4.0` requires tensorflow v1
